# Character

## Methods

```java
Character.isLetter(char c); // Checks for [A-z] and [a-z] and returns boolean

Character.isDigit(char c); // Checks if it is digit and returns boolean

Character.isUpperCase(char c); // Checks for upper case letter and returns boolean.
Character.toUpperCase(char c); // turns into upper case letter if char is passed, if ASCII value passed(int) then returns the ASCII value of upper case.
```
